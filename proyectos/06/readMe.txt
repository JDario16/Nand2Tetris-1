Compilacion:
g++ -std=c++11 *.cpp -o Assembler
O
haciendo uso del MakeFile con el comando make 

Ejecucion en consola:
./Assembler [archivo.asm]

el ejecutable puede ser usado arrastrando el .asm hacia el Assembler.exe
"abrir .asm con Assembler.exe"

Genera el .hack con el nombre dado en el archivo .asm

Ejemplo de ejecucion:
./Assembler Pong.asm

Ejemplo de salida:
Pong.hack
